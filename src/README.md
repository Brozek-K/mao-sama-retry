# 簡介

隨處可見的社會人士——大野晶在自己運營的遊戲內，以被稱之為「魔王」角色的登陸狀態下，轉移到了異世界。
<br /><br />
他在這裡遇到了一位右腳不便的小孩子，並一同啟程旅行。然而「魔王」持有壓倒性的力量，其他人不可能放任不管。
<br /><br />
他們被想要討伐魔王的國家和聖女所盯上，這場引發的騷動最終都把整座王國捲入其中。
<br /><br />
這是一部外觀上是魔王，可其內在是一般人的誤會系幻想世界。
